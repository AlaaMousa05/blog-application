# AlaaBlog

> A modern, interactive blog application built with vanilla JavaScript, featuring post management, commenting system, and an elegant user interface.

---

## 📋 Project Overview

**AlaaBlog** is a dynamic single-page application (SPA) that allows users to:
- Browse a collection of blog posts
- View detailed post information with comments
- Create new blog posts
- Delete posts and manage content
- Explore posts with a responsive and intuitive design

The application uses **Webpack** as a module bundler and fetches data from a REST API, featuring a clean modular architecture with proper separation of concerns.

---

## 🎯 Features

✅ **Post Listing** - Display all blog posts in a responsive grid layout  
✅ **Post Details** - View full post content with associated comments  
✅ **Create Posts** - Add new blog posts via an interactive form  
✅ **Delete Posts** - Remove unwanted posts from the blog  
✅ **Navigation Bar** - Dynamic navbar with page navigation  
✅ **Responsive Design** - Fully mobile-friendly interface  
✅ **Data Persistence** - LocalStorage integration for state management  
✅ **Modern Styling** - Beautiful gradient UI with smooth animations
